Promotional TPWARS gif

This is a promotional image of the platform, showing that the platform is also available on mobile

The original image can be found here. The origina download size was 14mb, the image dimensions were 5184px X 3456px with a resolution of 300px per inch. This is quite a large image for web. To create this gif I imported the mockups for the app into photoshop, and include them in different layers. To have more space for the text I have cloned the background at the top, using the clone tool. I also changed the resolution to 72 pixels per inch, as that is the standard web resolution and it's enought for this type of image, allowing me to have a reduced file size after exporting. To export, I have saved for web, as a gif with 256 colours, as this is a photography with different shades, I tried to mantain all the shades possible within the gif, while exporting I have also chosen perceptual to give priority to the colours that are seen by the human eye. I have also gave it a web snap of 40%, this converts the colours to the closest web colours. Finally, for the file size I have chosen to go with 1000 x 667px. as this is enough for a big screen, and it translates in a smaller file size than the original image size. The total output size was 454kb.
